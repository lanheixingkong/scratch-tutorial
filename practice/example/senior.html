# 5.基础口算练习

点击绿旗按钮开始游戏。

点击选择练习模式，20以内的加减法、两位数的加减法、100以内的乘除法。

点击游戏设置，可设定每局游戏的时间（默认30秒），自动切题时间（默认3秒）。

# 6.单位换算

点击绿旗按钮开始游戏。

点击按钮选择练习模式，“熟悉进率”或“单位换算”。点击小猫查看答案，自动进入下一题。

# 7.时光倒流

点击绿旗按钮开始游戏。

随着音乐拖动小螃蟹在舞台中任意移动，舞台中将留下小螃蟹移动过的痕迹。

音乐停止后，松开小螃蟹，随着音乐的反转，小螃蟹将自动沿着来时的路线时光倒流。
