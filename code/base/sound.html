# 将[音调]音效增加(10)

功能：增加或减少声音的音效，音效默认值为0。多个音效可以组合使用。作用于当前角色播放的声音。

音效类型参数：固定参数积木open in new window。选项为音调和左右平衡。

音调是声音的高低，数字越大，播放速度越快，音调越高，听着越尖锐刺耳。数字越小，播放速度越慢，音调越低，听着越低沉。

左右平衡是声音的左声道和右声道（如果你带着耳机就是左耳机和右耳机），数字越小，左声道越明显（右耳机声音变小），数字越大，右声道越明显（左耳机声音变小）。

数值参数：椭圆形参数open in new window，增加的音效数值。

返回值：无

难度：☆☆

演示视频open in new window
