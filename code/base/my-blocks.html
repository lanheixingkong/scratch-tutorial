# 自制积木

每个积木都是一堆指令的集合，它们各自实现了不同的功能。你可以将积木块想象成数学公式，为每个公式中带入不同的参数，再根据公式定义的规则进行计算，就可以得出需要的答案。公式中的计算规则，就是程序中的算法open in new window，属于对过程的抽象。

在Scratch中，可以将现有的积木块组合起来，使其成为一个实现特定功能的新积木块（函数）。使用自制积木最大的好处是将重复的指令串进行函数封装open in new window，既能减少程序的代码量，又能在函数功能发生改变时便于修改。

为自制积木取一个有意义的名字，还能极大的提高代码的可读性。
